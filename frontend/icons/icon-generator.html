✅ SVG Icons Available!
SVG icon files sudah tersedia di folder ini (icon-192.svg, icon-512.svg).
Anda dapat menggunakan tool online untuk convert SVG → PNG, atau ikuti langkah di bawah untuk generate sendiri.

Opsi 1: Gunakan Tool Online (Rekomendasi)

  1. Buka: Convertio SVG to PNG atau CloudConvert
  2. Upload file icon-192.svg atau icon-512.svg
  3. Download hasil PNG ke folder icons/

Opsi 2: Generate Dengan Canvas (langkah di bawah)

  1. Klik tombol untuk generate icon 192x192
  2. Right-click → Save image as icon-192.png ke folder icons/
  3. Generate icon 512x512
  4. Right-click → Save image as icon-512.png ke folder icons/
